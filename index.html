Submission Rules

Please read and abide by these content criteria rules strictly to secure fast monetization validation.

Rule 1: Minimum Views

Minimum 1K views required for clip submission.

Rule 2: Engagement Rates

Minimum 0.4% engagement required.
Example: For 1,000 views, at least 4 likes are required.

Rule 3: Creator Mentions

Caption must include the creator's tag/mention.

Rule 4: Geotargeting Location

Location must be set to any USA location (for example: Florida, California, Texas, New York, etc.).

Rule 5: Original Output Only

Only original, high-quality clips will be accepted.

Rule 6: Anti-Spam Clause

Spam, copied, or reused clips will be rejected.
